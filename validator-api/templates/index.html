Endpoints
POST /validate/iban
Validates any IBAN. Returns country, formatted version, check digits, BBAN. Supports 77 countries.
{ "iban": "GB82WEST12345698765432" }
POST /validate/vat
Validates EU VAT numbers. Returns country and normalized format. Covers all 27 EU states + UK.
{ "vat": "DE123456789" }
POST /validate/batch
Validate up to 100 IBANs and/or VATs in a single request. Returns per-item results + summary.
{ "ibans": ["GB82...", "DE89..."], "vats": ["DE123..."] }
GET /countries
Lists all supported countries with expected formats and lengths.
